# Ejercicio hecho en python

Este algoritmo se basa en el ingreso de un numero cualquiera simepre y cuando sea entero y positivo,
esto se logra ubicando una condición que diga que di es un numero menor o igual a 0 el sistema
diga que el número es inválido y que el usuario deberá ingresar nuevamente otro número.
Ahora si el número ya es entero positivo, este será calculado si es par o impar y arrojará el resultado.
